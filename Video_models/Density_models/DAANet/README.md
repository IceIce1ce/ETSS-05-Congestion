# DAANet

## 1. To install the environment, run the following script:
```shell
bash scripts/install.sh
```

## 2. To process the dataset, run the following script:
```shell
bash scripts/process_dataset.sh
```

## 3. To train, test, and visualize the model for SenseCrowd and CroHD datasets, run the following scripts:
```shell
bash scripts/train_sensecrowd.sh
bash scripts/train_crohd.sh
bash scripts/test_sensecrowd.sh
bash scripts/test_crohd.sh
bash scripts/test_speed.sh
bash scripts/vis_dataset.sh
```
